Purpose
This is a code package is related to the following scientific article:
K. Upadhya and S.A. Vorobyov, "Covariance matrix estimation for massive MIMO," IEEE Signal Processing Letters,  vol. 25, no. 4, pp. 546–550, Apr. 2018.
 

The package contains a simulation environment, based on Matlab, that reproduces all the numerical results and figures in the article. We encourage you to also perform reproducible research! Please run fig123.m to generate Figs. 1 - 3 in the paper and fig4.m to generate Fig. 4 in the paper.
